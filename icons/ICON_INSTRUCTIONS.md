# Icon Creation Instructions

Since we cannot generate actual PNG files programmatically, you'll need to create icon files manually or use an online icon generator.

## Required Icons:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Quick Way to Create Icons:

1. **Online Icon Generator**: 
   - Visit https://favicon.io/favicon-generator/ or similar
   - Create a simple icon with "YT" text or a YouTube-related symbol
   - Download in multiple sizes

2. **Use Existing Icons**:
   - Find a suitable icon from https://icons8.com or similar
   - Download in required sizes

3. **Simple Design Recommendation**:
   - Background: Clean blue/purple gradient
   - Symbol: "YT" text in white or a crossed-out ad symbol
   - Keep it simple and recognizable

## Alternative: Use Base64 Encoded Icons

If you want to avoid creating separate files, you can use data URLs in the manifest, but separate files are recommended for better performance.

Place all icon files in the 'icons' folder:
- icons/icon16.png
- icons/icon32.png
- icons/icon48.png
- icons/icon128.png
